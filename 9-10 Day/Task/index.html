HTML & CSS Basics Summary

HTML Summary

HTML is used to define our page content, structure and meaning. You don't use it styling purposes. Use CSS for that instead!

  • HTML uses "elements" to describe (annotate) content
  • HTMl clements typically have an opening tag, content and then a closing tag
  • You can also have void(empty) elements like images
  • You can also configure elements with attributes
  • There's more list of available elements but you'll gain experience over time, no worries.

    • Learn more about all available HTML elements on the MDN HTML element Reference
